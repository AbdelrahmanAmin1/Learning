تشغيل المشروع:

1) افتح الطرفية داخل مجلد `arabic-kids-site`
2) شغّل الأمر:
   `python server.py`
3) افتح المتصفح على:
   `http://localhost:8000`

ما الذي أصبح موجودًا الآن:

- خادم Python فعلي يخدم الواجهة وواجهات API
- قاعدة بيانات SQLite تُنشأ تلقائيًا في أول تشغيل
- تسجيل طالب جديد وتسجيل دخول عبر الباكند
- لوحة طالب لعرض المستوى والإنجاز والحضور
- لوحة معلم لإدارة الطلاب والتقدم والحضور
- نشر روابط زوم والمواد التعليمية لكل مستوى
- قسم تواصل يحفظ الرسائل في قاعدة البيانات

ملاحظات:

- المحتوى العام للمستويات والكتب يُزرع تلقائيًا داخل قاعدة البيانات
- بيانات الطلاب والجلسات والحضور تبدأ فارغة، بدون حسابات طلابية وهمية
- يتم إنشاء حساب معلم محلي تلقائيًا في أول تشغيل
- يمكن تغيير بريد وكلمة مرور المعلم عبر متغيرات البيئة:
  `ALKTTAB_TEACHER_EMAIL`
  `ALKTTAB_TEACHER_PASSWORD`
